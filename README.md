#  LyricMotion

##  Descripción

**LyricMotion** es una aplicación Android desarrollada en Kotlin que permite visualizar letras de canciones de forma dinámica e interactiva. Está diseñada para ejecutarse en dispositivos modernos con soporte para la API 35.

---

## Tecnologías utilizadas

* Lenguaje: Kotlin
* IDE: Android Studio (última versión)
* Plataforma: Android

---

## Clonar el repositorio

Para obtener una copia del proyecto en tu máquina local, ejecuta:

```bash
git clone https://github.com/JXANX/LyricMotion.git
```

Luego entra a la carpeta del proyecto:

```bash
cd LyricMotion
```

---

## Requisitos

Antes de ejecutar la aplicación, asegúrate de cumplir con lo siguiente:

* Tener instalado Android Studio (última versión)
* Tener configurado el SDK de Android
* Dispositivo o emulador con:

  * **Android API 35**

---

## Compilación y ejecución

1. Abre Android Studio
2. Selecciona **"Open"** y abre la carpeta del proyecto
3. Espera a que Gradle sincronice automáticamente
4. Conecta un dispositivo físico o inicia un emulador con **API 35**
5. Haz clic en el botón **Run **

La aplicación se compilará e instalará automáticamente en el dispositivo seleccionado.
